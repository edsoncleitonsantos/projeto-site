A HTMLé uma linguagem de marcação para a criação de sites. Atualmente na versão 5, ela usa a semântica dos elementos para criar o conteúdo de uma página Web. Com as CSS, criamos as configurações visuais que um site vai ter.

Curos de JavaScript

JavaScripté uma linguagem de script criada para aumentar a interatividade entre um site e seus visistantes. Os códigos em JS são executados dentro do navegador e dão acesso aos elementos HTML por meio do DOM.
